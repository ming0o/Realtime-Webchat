# Development Guide

## ✅ Prerequisites
- Node.js v18 이상
- Android Studio / Xcode 설치 (RN 개발용)

## ⚡️ 개발 절차
1. 저장소 클론
    ```
    git clone https://github.com/.../realtime-webchat.git
    ```
2. Backend 세팅
    ```
    cd backend
    npm install
    npm run dev
    ```
3. Frontend (web) 세팅
    ```
    cd frontend/web
    npm install
    npm run dev
    ```
4. Frontend (mobile) 세팅
    ```
    cd frontend/mobile
    npm install
    npx react-native run-android
    ```
